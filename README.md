## Run Locally  

Clone the project  

~~~bash  
  git clone https://github.com/Mohabz-911/TextEditor
~~~

Go to the project directory  

~~~bash  
  cd TextEditor
~~~

Install python

Start the program  

~~~bash  
py .\TextEditor.py
~~~
